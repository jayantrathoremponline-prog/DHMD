# DHMD
# DHMD stands for Digital Height Measurement Device
A real‑time height measurement system using an HC‑SR04 ultrasonic sensor, HC‑05 Bluetooth, and MIT App Inventor.

## Features
- Accurate height measurement with 98 % accuracy.
- Bluetooth control (Calibrate, Pause, Resume)
- MIT App Inventor interface for live monitoring
- Moving average filter for stable readings

## Hardware
- Arduino Uno/Nano
- HC‑SR04 Ultrasonic Sensor
- HC‑05 Bluetooth Module
- 5 v constant power supply

## Setup
1. Wire as per the diagram
2. Upload Arduino code
3. Install the DHMD App
4. Connect via Bluetooth
